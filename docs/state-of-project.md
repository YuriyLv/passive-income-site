## Наступні кроки
1. Увімкнути Google Search Console API.
2. Створити або заповнити JSON-ключ сервісного акаунта та перезапустити `bash scripts/indexing-check.sh`.
3. Створити/заповнити API-ключ Bing Webmaster.
4. Заповнити `docs/partner-profiles/` після надання доступу до партнерських порталів.
5. Активувати браузер для реєстрації/підтвердження соцмережних профілів.
