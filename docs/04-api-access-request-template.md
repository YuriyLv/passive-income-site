Шановний(-а) [ім’я]!

На проєкті [BizTool Insight](https://biztoolinsight.com.ua/)
заплановано автоматизовані щотижневі перевірки Coverage/Scan у
Google Search Console та Bing Webmaster. Для цього необхідний програмний
доступ:

### Що потрібно надати

1. **Google Search Console**
   - додати сервісний обліковий запис як користувача/редактора сайту
     `https://biztoolinsight.com.ua/`;
   - увімкнути API **Google Search Console API** у Google Cloud;
   - надати JSON-файл ключа сервісного акаунта із правами
     `https://www.googleapis.com/auth/webmasters`.

2. **Bing Webmaster**
   - додати користуваря/співпрацівника для даного сайту;
   - створити API-ключ в Bing Webmaster і надати його значення.

### Що буде здійснено після надання доступу

- тестовий запит для перевірки роботи ключів;
- перший офіційний запис результатів у протоколі `docs/indexing-protocol.md`;
- запланована автоматизована перевірка кожного тижня о 03:00 через `cron/indexing-weekly.sh`.

---

Замінити заповнення значень у загальному доступі запобігать:
- конфіденційні ключі зберігатимуться тільки у `.env` за шаблоном `.env.example`;
- доступ надаються лише на конкретні сервіси і можуть бути скасовані у будь-який момент.

Будь ласка, підтвердіть можливість надання доступу та подальші терміни.
Якщо виникнуть обмеження — готовий скоригувати спосіб інтеграції.
