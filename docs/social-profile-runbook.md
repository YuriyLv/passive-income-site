# Social Profile Runbook

Цей документ фіксує план соціальних профілів для подальшого виконання, поки автоматичний вхід обмежений.

Попередня перевіра: чи доступна у середовищі браузерна автоматизація/X/Twitter API для створення профілю без вручного входу.

## Актуальні профілі

### X / Twitter
- Мета: запуск основного профілю бренду
- Дії: створити обліковий запис, заповнити біо, додати посилання на сайт, опублікувати перший пост
- Блокування зараз: браузерна автоматизація не налаштована
- Наступний крок: активувати Browser Use, якщо це можливо

## Виконані на поточному запуску дії

- Зарахували окремий runbook для соцпрофілів
- Готовий план для майбутнього запуску облікових записів після активації браузера
