# Профілі партнерських программ

## Amazon Associates

- **Назва партнерської програми:** Amazon Associates
- **Посилання на програму:** https://affiliate-program.amazon.com/
- **Тип винагороди:** Відсоток від продажу / CPS
- **Статус участі:** Заплановано до реєстрації
- **Літ-ланд / умови:** https://affiliate-program.amazon.com/help
- **Кабінет партнера:** https://affiliate-program.amazon.com/
- **Обліковий запис проєкту в програмі:** Не створено
- **Формат посилання:** Стандартні афіліа́тні посилання через `amzn.to` або повні URL з тегом `tag=...`
- **Рекомендовані категорії:** Бізнес-інструменти, книги про продуктивність та пасивний дохід, гаджети, програмне забезпечення
- **Первинна оцінка:** Високий потенціал, оскільки сайт присвячений бізнес-інструментам та пасивному доходу.
- **Заплановані статті:** "7 інструментів для автоматизації малого бізнесу", "Книги, які змінили мій підхід до доходу", "Наочні приклади ефективного трекінгу часу"
- **Ліміти / умови:** Комісії від 1% до 10% залежно від категорії. Виплати щомісяця.
- **Контакт підтримки:** https://affiliate-program.amazon.com/help
