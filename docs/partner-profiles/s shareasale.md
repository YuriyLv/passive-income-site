# Профіль партнерської програми: ShareASale

## Ідентифікація

- **Назва партнерської програми:** ShareASale
- **Посилання на програму:** https://www.shareasale.com/
- **Тип винагороди:** Відсоток від продажу / CPS, а також фіксовані винагороди
- **Статус участі:** Заплановано до реєстрації

## Посилання для моніторингу

- Літ-ланд / умови: https://www.shareasale.com/information/writers-wanted.htm
- Кабінет партнера: https://www.shareasale.com/
- Обліковий запис проєкту в програмі: Не створено

## Параметри для вставки у контент

- Мінімальний відсоток реферальних посилань у тексті: Збалансований контент, максимум 1 релевантне посилання на статтю
- Формат посилання, дозволений програмою: Створення унікальних deep- links через кабінет партнера
- Категорії товарів/послуг, рекомендованих для просування: Бізнес-продукти, освітні курси, маркетингові інструменти, SaaS

## Примітки проєкту

- Первинна оцінка відповідності сайту: Висока релевантність через велику кількість бізнес- і освітніх мереж.
- Заплановані статті, де планується використання:
  - "Як створити онлайн-курс: порівняння платформ"
  - "Маркетингові інструменти, які варто випробувати"
- Ліміти на UM / спеціальні умови встановлення: Термін виплат змінюється мережами, зазвичай щомісяця.
