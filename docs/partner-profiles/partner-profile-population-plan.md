# План заповнення профілів партнерських програм

Цей документ визначає джерела даних для `docs/partner-profiles/affiliate-program-profile.md`, щоб уникнути вигаданих цифр/посилань.

## Джерела даних

1. `docs/partner-profiles/teachable-affiliate.md` — вже існує частковий профіль для Teachable.
2. `affiliate-programs.md` — загальний перелік партнерських програм.
3. `account-setup-guide.md` — URL реєстрації та статус створення облікових записів.
4. `affiliate-placeholders/*.md` — деталі для конкретних програм.
5. `docs/indexing-protocol.md` та `docs/social-profiles.md` — контекст поточних проєктних обмежень.

## Підзадачі перед заповненням профілів

- [ ] Перевірити, чи є у проєкті реальні профілі партнерських програм у `docs/partner-profiles/` окрім `teachable-affiliate.md`.
- [ ] Зібрати фактичні URL реєстрації та назви програм у `docs/partner-profiles/program-inventory.md`.
- [ ] Оновити `affiliate-placeholders/*.md` на підставі реальних програм з `affiliate-programs.md`.
- [ ] Заповнити `docs/partner-profiles/affiliate-program-profile.md` тільки перевіреними даними з вищезазначених файлів.

## Правила

- Не вказувати фіктивні посилання, прибуток, контакти або ідентифікатори.
- Якщо даних бракує, залишати поле порожнім або з тегом `TODO`.
