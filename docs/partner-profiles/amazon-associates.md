# Профіль партнерської програми: Amazon Associates

## Ідентифікація

- **Назва партнерської програми:** Amazon Associates
- **Посилання на програму:** https://affiliate-program.amazon.com/
- **Тип винагороди:** Відсоток від продажу (комісія)
- **Статус участі:** Заплановано до реєстрації

## Посилання для моніторингу

- Літ-ланд / умови: https://affiliate-program.amazon.com/help
- Кабінет партнера: https://affiliate-program.amazon.com/
- Обліковий запис проєкту в програмі: Не створено

## Параметри для вставки у контент

- Мінімальний відсоток реферальних посилань у тексті: Домінуючий контент має мати релевантні посилання на продукти
- Формат посилання, дозволений програмою: Стандартні афіліа́тні посилання через `amzn.to` або повні URL з тегом `tag=...`
- Категорії товарів/послуг, рекомендованих для просування: Бізнес-інструменти, книги про продуктивність та пасивний дохід, гаджети, програмне забезпечення

## Примітки проєкту

- Первинна оцінка відповідності сайту: Високий потенціал, так як сайт присвячений бізнес-інструментам та пасивному доходу.
- Заплановані статті, де планується використання:
  - "7 інструментів для автоматизації малого бізнесу"
  - "Книги, які змінили мій підхід до доходу"
  - "Наочні приклади ефективного трекінгу часу"
- Ліміти на UM / спеціальні умови встановлення: Комісії від 1% до 10% залежно від категорії. Виплати щомісяця.
- Контакт підтримки програми: https://affiliate-program.amazon.com/help
