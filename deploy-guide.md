# Інструкція з деплою
## GitHub Pages
1. Створити репозиторій `biztoolinsight`
2. Завантажити файли сайту
3. У налаштуваннях репозиторію ввімкнути Pages, гілка `main`
4. Отримати доступний URL виду `https://biztoolinsight.github.io/biztoolinsight`

## Netlify
1. Створити акаунт на Netlify
2. Підключити репозиторій GitHub
3. Публікувати папку з файлами сайту
4. Налаштувати кастомний домен (за потреби)
