# Інструкція з реєстрації партнерських облікових записів

Послідовність по 3-5 ключових програм, оптимізована під український/світовий ринок.

## Пріоритет №1: Amazon Associates
- URL реєстрації: https://affiliate-program.amazon.com/
- Потрібні дані: Ім'я, країна (UA), вебсайт, мінімальний опис трафіку.
- Важливо: UA-домен приймається, але can withdraw to US bank / Payoneer. Перевірте актуальні умови для UA.
- Додайте в імені фразу "BizTool Insight".

## Пріоритет №2: CJ Affiliate
- URL реєстрації: https://www.cj.com/affiliates
- Потрібні дані: вебсайт, назва компанії, контакт.
- Способи виплат: PayPal, ACH для US; для UA — уточніть в портфолі облікового запису/партнерському менеджеру.
- Шаблон листа: "We run BizTool Insight, a site focused on passive income tools and guides. Expect traffic from organic search."

## Пріоритет №3: Teachable (якщо тренери/курси підходять)
- URL: https://teachable.com/affiliates
- Запит: fill partner form; очікуйте approval.

## Пріоритет №4: TradingView
- URL: https://tradingview.com/affiliate/
- Ключові дані: реферальний ресурс, соцмережі.
- Умови: дозволено в UA, але перевірте юридичні обмеження.
- Виплати: PayPal / крипто в залежності від програми.

## Пріоритет №5: Українські сервіси
- Prom.ua: https://prom.ua/partnership
- Rozetka: партнерська програма через Partner Network/Impact або безпосередній email.
- Дивіться вплив в UA-ринку + CPA модель.

## Шаблон листа для менеджера/підтримки
Subject: Partnership Application — BizTool Insight
Body:
"Hello,
We are launching BizTool Insight, a resource covering passive income tools, affiliate marketing and productivity guides.
We plan to promote relevant offers via review articles and comparison tables.
Could you please confirm the application thread and Territory/Commissions for UA traffic?
Thank you,
[Your name]"

# Процес
1. Обрати першу програму (Amazon Associates або CJ Affiliate).
2. Заповнити анкету, використовуючи шаблон вище.
3. Очікувати approve; додати партнерські посилання в /home/deploy/projects/passive-income-site/affiliate-placeholders.
4. Повторити з наступними програмами.
