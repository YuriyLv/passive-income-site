# Наступні кроки

1. Заповнити `.env` реальними ключами: `GSC_SITE_URL`/`GSC_KEY_PATH`/`BING_WEBMASTER_API_KEY`/`BING_SITE_URL` та перезапустити `scripts/indexing-check.sh`.
2. Виконати налаштування HTTPS через `docs/06-https-letsencrypt-setup.md` (запит/активація сертифіката для `biztoolinsight.com.ua`).
3. Активувати доступ до зовнішнього браузера/Browser Use та виконати планове створення соцмережних профілів (`docs/social-account-create-runbook.md`, `docs/social-browser-setup-workarounds.md`).
4. Заповнити `docs/partner-profiles/` фактичними даними партнерських програм після отримання доступу до порталів.
5. Після створення реального GA-ресурсу замінити в `index.html` заглушку `G-XXXXXXXXXX` на діючий ідентифікатор.
6. Додати реферальні посилання в решту статей, де її ще немає.

Ключовий блокер наразі — відсутність реальних секретів у `.env` для індексації та відсутність активованого браузера для соцмереж.
