# Deploy checklist

## Before publishing
1. Замінити `https://example.com/` у канонічних посиланнях і sitemap на реальний домен.
2. Переконатися, що репозиторій імʼя та протокол збігаються з обраною платформою.
3. Додати Google Search Console / Bing Webmaster. Verify властивостей.
4. Підключити сторінки в партнерських програмах; додати відповідні посилання в статтях.
5. Оновити `hosts.txt` тут, якщо використовуєш статичний хост.
6. Запустити локальну перевірку: `python3 -m http.server 8080` -> відкрити головну і пости.

## Netlify quick start
1. Створити сайт на Netlify.
2. Підключити репозиторій або вручну надати папку `/home/deploy/projects/passive-income-site`.
3. Вказати корінь сайту: `public` або корінь проєкту для статичного сайту.
4. У ручному режимі завантажити зібрану папку.
