# Пасивний дохід — сайт-огляд SaaS для бізнесу

## Мета
Створити невеликий контентний сайт з партнерськими посиланнями, щоб з часом отримувати регулярний дохід без постійної активної роботи.

## План
- Сайт на GitHub Pages / Netlify
- Контент: рекомендації з вибору EDO, POS, CRM, бухгалтерських сервісів
- Дохід через партнерські програми
- Розвиток: 1-2 статті на тиждень

## Деплой
- GitHub Pages: налаштувати через налаштування репозиторію
- Netlify: підключити репозиторій, вказати папку з файлами

## Доступ до доходу
- Реєстрація в партнерських програмах
- Виплати йдуть на вказаний платіжний рахунок
- Перші виплати зазвичай через 30-90 днів після отримання замовлення
